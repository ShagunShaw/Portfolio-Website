QuizBlitz — Real-Time Quiz Hosting Platform

GitHub | Live Link (on Render/Vercel)

TypeScript • Node.js • Express.js • Socket.io • MongoDB • JWT • Nodemailer • Winston

Developed a scalable real-time quiz hosting platform capable of supporting multiple concurrent quiz sessions with synchronized gameplay and server-side state management.

SecureDrop — Ephemeral File Sharing Platform

Github

NestJS • TypeScript • MongoDB • Cloudinary • HTML • CSS • JavaScript

Developed a secure burn-after-reading file sharing platform designed for temporary and privacy-focused file exchange using unique access codes and automatic expiry mechanisms.

Architected a modular NestJS backend and built a custom upload processing pipeline using Multer memory storage and Archiver to automatically aggregate multiple uploaded files into a single ZIP bundle before cloud storage.

Implemented scheduled cleanup services using @nestjs/schedule that run nightly to permanently remove expired files from Cloudinary and delete associated metadata records from MongoDB.

Strengthened platform security through custom validation pipes, restricted file-type detection, upload-size limitations and API rate limiting using @nestjs/throttler to prevent abuse and unauthorized uploads.
